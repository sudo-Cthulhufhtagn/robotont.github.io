Controlling the robot¶

  1. The robot driver subscribes to a specific type of messages called velocity commands. The standard name for this topic is /cmd_vel or /robotont/cmd_vel when in robot namespace.
  2. The message is of type geometry_msgs/Twist and it’s structure can be found from ROS wiki.
  3. To set and control the robot speed, the velocity commands need to be published continuously.

Setup¶

  1. Install teleop twist keyboard

    sudo apt update
sudo apt install ros-melodic-teleop-twist-keyboard

  2. From your Android device, go to Google Play Store and install the ROS Control app.

  3. Establish an ssh connection between the robot and the user PC as shown here: Connecting with the robot remotely.

Controlling the robot using teleop twist keyboard¶

  1. Open a new terminal window

  2. On ROBOTONT on-board computer run the following command:

    rosrun teleop_twist_keyboard teleop_twist_keyboard.py __ns:=/robotont

    or

    roslaunch robotont_demos teleop_keyboard.launch

  3. Use the following keys to move the robot:

    Warning

    From this point beyond, you are able to drive the robot with a keyboard. Should you loose control over the robot, do one of the following

    • PRESS “k” TO STOP THE ROBOT!
    • PRESS THE EMERGENCY SWITCH ON THE ROBOT.

    Hint

    Notice that the teleop node only receives keypresses only when the terminal window is active.

    Tip

    Use CTRL + C to stop the node.

Controlling the robot using an Android device¶

  1. Open the ROS Control app on your phone

  2. Insert the ROBOTONT’s IP address into Master URI field by entering the following:

    http://robotont_IP_address:11311

  3. Click on “Show advanced options” in the prompted window and fill in “Joystick” and “Odometry” topic names with “robotont/cmd_vel” and “robotont/odom”, respectively

  4. Click OK to add the robot

  5. Now you can select the robot from the list and teleoperate it using the touch joystick button
