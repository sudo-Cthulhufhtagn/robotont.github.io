Controlling the simulated robot on RViz¶

Setup¶

  1. Install teleop twist keyboard

    sudo apt update
sudo apt install ros-melodic-teleop-twist-keyboard

  2. From your Android device, go to Google Play Store and install the ROS Control app.

  3. Start the driver

    roslaunch robotont_driver fake_driver.launch

  4. Make sure that fixed frame is set to “odom”!

Controlling the robot using teleop twist keyboard¶

  1. Open a new terminal window

  2. Run the following command:

    rosrun teleop_twist_keyboard teleop_twist_keyboard.py

  3. Use the following keys to move the robot:

    Hint

    Notice that the teleop node receives keypresses only when the terminal window is active.

    Tip

    Use CTRL + C to stop the node.

Controlling the robot using an Android device¶

  1. Make sure that the user PC and Android device are connected to the same wifi router

  2. Open the ROS Control app on your phone

  3. Insert your computer’s IP address into Master URI field by entering the following:

    http://IP_address:11311

  4. Click on “Show advanced options” in the prompted window and fill in “Joystick” and “Odometry” topic names with “cmd_vel” and “odom”, respectively

  5. Click OK to add the robot

  6. Now you can select the robot from the list and teleoperate it using the touch joystick button
